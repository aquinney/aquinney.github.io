Site Name

The site name I have chosen is "What's the weather like?"

Site Purpose

The purpose of this website is to allow users to discover as the name implies, what the weather will be like at a given location and at a given time. The main goal of the website is to make it as simple to use as possible. That way, even inexperienced users can easily figure out where to go to find what they're looking for.

Domain Name

The domain name would be "whatstheweatherlike.com"

Site Logo

The logo will be a simple "sky blue" circle with a cloud in the middle. This logo was obtained from "iconfinder.com"
